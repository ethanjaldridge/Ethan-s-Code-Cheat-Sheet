Emmet
Tag Name Description
Emmet A collection of shortcuts and snippets that generate long-hand code in HTML.
! + Enter Creates boilerplate for an HTML file.
Tags Emmet is also responsible for generating a full tag from even a single letter (i.e. "a" for "anchor").
tag.class Generates a tag with the desired class.
tag#id Generates a tag with the desired id.
.class / #id Tags without a name will default to a <div> tag.
tag[attribute="value"] / tag[attribute] Generates a tag with the desired attribute.
tag>child Generates a tag with the desired child element.
tag*number Generates the desired tag, a desired number of times.
tag{HTML text here} Generates a tag with the desired text between the element's tags.
tag*3{$} / .class-$ The "$" sign will generate numbers in its place.
tag*3{$$} Using 2 "$" signs will pad the number, adding a 0 in front of it.
header+main+footer The "+" symbol will generate sibling tags (as it does in css).
header>nav^main+footer The "^" symbol serves to "climb up" out of the header tag, and continue creating sibling tags.
(header>nav)+main+header The "()" symbols "group" the tags together, so that the sibling tags aren't nested within the header.
input:text Creates an input with the "type" of text.
